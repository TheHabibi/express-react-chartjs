# Graph Application using an Express Server and Chart.js
## Used Technologies
- Node.js
- Express.js
- React
- Chart.js

## How to Install
- clone the project
- Get inside the "lonca-case" director
- "npm install"
- "npm start"
- "cd frontend"
- "npm install"
- "npm start"
- Open http://localhost:3000/ on your browser
